Projeto de conclusão de curso

Esse projeto foi desenvolvido como requisito parcial para obtenção do grau de bacharel de Sistemas de Informação

O objetivo desse trabalho é desenvolver uma ferramenta para avaliação preditiva de interface, utilizando o método de avaliação por checklist, possibilitando o cadastro de questionários personalizados e a geração de relatorios.

O projeto foi desenvolvido na linguagem Python, utilizando o framework Django

Observação:
O projeto utiliza o Django 1.10.6 e suas bibliotecas nativas.
